# Escolas

- Brazil has more than 150,000 schools. <br/>
This demo App plot the location around the country. <br/>
There are some filters such as type (private, public, both, state and city.<br/>
In the future, it will be added some objectives and results grouped by school.<br/>
You can see the App running at: https://share.streamlit.io/humbertomoura/escolas/main/main.py
