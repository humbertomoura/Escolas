# Escolas

- Brazil has more than 150,000 schools.
This demo App plot the location around the country. 
There are some filters such as type (private, public, both, state and city.
In the future, it will be added some objectives and results grouped by school.
You can see the App running at: https://share.streamlit.io/humbertomoura/escolas/main/main.py
